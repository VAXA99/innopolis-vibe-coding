# 🔄 User Flow

## 🌙 Вечерний сценарий (основной)

1. Пользователь открывает приложение  
2. Попадает на главный экран  
3. Устанавливает время будильника  
4. Настраивает окно пробуждения  
5. Выбирает звук для сна  
6. Настраивает таймер отключения  
7. Нажимает “Start Sleep”  
8. Приложение переходит в режим сна  

---

## ⏰ Утренний сценарий

1. Будильник срабатывает в заданном окне  
2. Происходит мягкое пробуждение  
3. Пользователь видит экран с результатами:
   - длительность сна  
   - рекомендация  

---

## 📊 Дневной сценарий (опционально)

1. Пользователь открывает приложение  
2. Переходит в раздел статистики  
3. Просматривает:
   - данные о сне  
   - уровень активности  
4. Получает инсайты и рекомендации
